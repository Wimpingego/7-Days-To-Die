# Ego UI

Small UI edit to add food and water.

## What this Modlet does

This “Modlet” adds water and food infor to the UI.


## Installing

Copy the **Ego UI** folder into the 7D2D Mods folder.

```
\SteamLibrary\steamapps\common\7 Days To Die\Mods
```

You may need to create the “Mods” folder if you don’t already have one.
